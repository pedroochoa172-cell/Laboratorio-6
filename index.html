Laboratorio 6

En este trabajo utilizaremos Git como herramienta de control de versiones para mantener organizadas y seguras las distintas versiones de nuestra página web. Usar Git nos permitirá registrar cada cambio que hagamos y trabajar de manera ordenada, lo que es muy útil en el desarrollo de sitios web.

En las siguientes páginas hablaré sobre cosas que me gustan, temas que me interesan y lugares que me gustaría visitar. A través de este laboratorio, combinaré el uso de Git con la creación de contenido personal en una página web.

Git es un sistema de control de versiones creado por Linus Torvalds. Permite llevar un registro de todos los cambios realizados en los archivos de un proyecto, facilitando la colaboración entre programadores y el control del desarrollo. Es una herramienta esencial para cualquier desarrollador moderno.
